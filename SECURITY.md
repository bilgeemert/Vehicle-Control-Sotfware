# Security Policy
## Supported Versions

We release patches for security vulnerabilities in the following versions of Secure-Drive-Vehicle:

| Version   | Supported          |
| --------- | ------------------ |
| 1.x       | :white_check_mark: |
| < 1.0     | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please follow these steps:
1. **Use the GitHub Security Advisories feature**:
    - Navigate to the [Secure-Drive-Vehicle repository](https://github.com/serkanMzlm/Secure-Drive-Vehicle).
    - Click on the "Security" tab.
    - Select "Report a vulnerability".
    - Follow the prompts to provide details about the vulnerability.
2. Provide as much detail as possible about the vulnerability so that we can reproduce and fix it as quickly as possible. Include:
    - A description of the vulnerability.
    - Steps to reproduce the issue.
    - Any relevant proof of concept, exploit code, or screenshots.
3. We will acknowledge your report within 48 hours and provide a detailed response within 7 days indicating the next steps in handling your report.
